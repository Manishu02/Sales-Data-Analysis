# 📊 Sales Data Analysis Project

## 📌 Project Overview
This project focuses on analyzing sales data to extract meaningful business insights such as revenue trends, top-performing countries, cities, and product lines.

The goal is to perform Exploratory Data Analysis (EDA) using Python and visualize key business patterns.

---

## 📁 Dataset Information

The dataset contains transactional sales data with the following key attributes:

- Order details (Order number, Order line number)
- Order date
- Sales value
- Quantity ordered
- Price per unit (MSRP)
- Product information (Product line, Product code)
- Customer details (Customer name, Country, City, State, Postal Code)
- Deal size (Small, Medium, Large)
- Territory and region information


---

## 🧹 Data Cleaning
- Converted ORDERDATE column to datetime format  
- Handled missing values using fillna()  
  - Numeric values filled with median  
  - Categorical values filled with "Unknown"  
- Converted SALES column to numeric format  

---

## 📊 Key Analysis Performed

### 📅 Monthly Sales Trend
Analyzed sales distribution over months to identify seasonal patterns.

### 🌍 Top 5 Countries by Sales
Identified countries contributing highest revenue.

### 🏙️ Top 5 Cities by Sales
Found cities with highest sales performance.

### 📦 Product Line Analysis
Analyzed which product categories perform best.

### 💳 Deal Size Analysis
Studied revenue contribution by deal size (Small, Medium, Large).

### 📈 Sales Heatmap
Visualized monthly vs yearly sales trends.
